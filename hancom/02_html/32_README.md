# 개발자도구 (F12) — 화면 뒤를 들여다보는 꿀팁

브라우저에 숨어 있는 개발자도구는 화면 뒤 HTML·CSS를 모두 볼 수 있고, 값을 임시로 변경해 미리보기도 가능한 에러 잡는 실전 도구입니다. 별도 설치가 필요하지 않으며 크롬, 엣지에 기본 내장되어 있습니다.

## 여는 법

- **F12** 키 누르기 (Mac은 **Cmd + Opt + I**)
- 또는 화면 아무 곳에서 우클릭 → **검사**

## 사용 방법

### 1. 검사 아이콘 활용
화살표(검사) 아이콘을 클릭한 후, 화면 요소에 마우스를 올리면 해당 태그가 무엇인지 정확히 확인할 수 있습니다.

### 2. Elements 탭
페이지의 HTML 구조를 확인할 수 있습니다. 특정 줄을 클릭하면 화면에서 그 부분이 반짝 표시됩니다.

### 3. 스타일 미리보기
글자나 색 값을 더블클릭해 임시로 변경하면, 화면에 바로 반영되어 디자인을 미리볼 수 있습니다.

### 4. Console 탭
빨간 글씨로 표시된 에러 메시지를 확인할 수 있으며, 어디가 잘못되었는지 힌트를 얻을 수 있습니다.

## ⚠️ 주의사항

여기서 변경한 내용은 화면에서만 임시로 반영됩니다. 실제 파일은 변경되지 않으며, 새로고침하면 원래대로 돌아옵니다. 따라서 안심하고 자유롭게 실험해볼 수 있습니다.

## 💡 학습 팁

남의 잘 만든 사이트에서 F12를 열어 구조를 살펴보는 것이 가장 빠른 공부 방법입니다.
