# Claude Code 설치 및 활성화

1. 웹 브라우저에서 Claude Code 설치 페이지로 접속한다.
   https://code.claude.com/docs/ko/quickstart#native-install-recommended

2. 터미널(파워쉘)에서 다음 명령어를 실행해서 Claude Code를 설치한다.
   irm https://claude.ai/install.ps1 | iex

- 윈도우 기본 명령 프롬프트 대신, Microsoft Store 에서 '터미널'을 설치해서 사용하는 것을 추천함.
  https://apps.microsoft.com/detail/9N0DX20HK701?hl=ko-kr&gl=KR&ocid=pdpshare

3. 터미널에서 `claude` 를 입력 후 실행하여 활성화
